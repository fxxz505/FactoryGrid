# FactoryGrid 多层生产与研究中心扩展设计

## 1. 目标

本扩展把生产、物流和研究统一为一个可持续扩张的闭环：玩家先建立矿物与基础零件产线，再制造不同层级的研究包，将研究包送入研究中心。研究中心只有在双击选择具体项目后才会消耗对应研究包，完成项目后解锁新配方、物流能力和机器等级。

系统继续遵循中文界面、Canvas 2D、简单几何体、固定 tick 模拟、机器与传送带无缝连接的项目原则。

## 2. 层级图案规则

工业物品不再全部使用三角形。复杂度每提升一级，几何图案增加一条边，让玩家不依赖文字也能识别物品等级。

| 层级 | 图案 | 典型物品 |
| --- | --- | --- |
| T0 | 圆形 | 铁矿、铜矿、煤矿 |
| T1 | 正三角形 | 铁锭、铜锭 |
| T2 | 正方形 | 铁板、钢材、齿轮、铜线、轴承 |
| T3 | 正五边形 | 电路、钢制框架、物流研究包 |
| T4 | 正六边形 | 电机、处理器、自动化/冶金/电子研究包 |
| T5 | 正七边形 | 伺服机构、自动化核心、机器人/核心研究包 |
| T6 | 正八边形 | 效用研究包 |
| Tn | 正 `n+2` 边形 | 未来新增更高层级物品 |

图案由物品数据中的 `tier` 字段驱动。Canvas、合成器配方面板和研究中心面板共用同一规则，避免同一物品在不同界面出现不同图案。

## 3. 完整生产闭环

```mermaid
flowchart LR
  Ore["T0 矿物"] --> Ingot["T1 金属锭"]
  Ingot --> Basic["T2 铁板/铜线/齿轮/钢材"]
  Basic --> Circuit["T3 电路/钢制框架"]
  Circuit --> Motor["T4 电机/处理器"]
  Motor --> Servo["T5 伺服机构/自动化核心"]
  Servo --> Utility["T6 效用研究包"]
  Basic --> Packs["T3-T5 研究包"]
  Packs --> Lab["研究中心"]
  Lab --> Unlock["配方/高速物流/机器等级"]
  Unlock --> Circuit
```

### 3.1 基础冶金

- 铁矿 + 煤矿 -> 熔炉 -> 铁锭。
- 铜矿 + 煤矿 -> 熔炉 -> 铜锭。
- 铁锭 + 煤矿 -> 熔炉 -> 钢材。

### 3.2 基础加工

- 铁锭 -> 铁板。
- 铜锭 -> 铜线 x2。
- 铁板 x2 -> 齿轮。
- 铁板 + 铜线 x2 -> 电路。

### 3.3 高级机械与电子

- 钢材 -> 轴承。
- 钢材 x2 + 铁板 x2 -> 钢制框架。
- 齿轮 + 电路 -> 电机。
- 电路 x2 + 铜线 x2 -> 处理器。
- 电机 + 电路 + 轴承 -> 伺服机构。
- 处理器 + 钢制框架 + 伺服机构 -> 自动化核心。

### 3.4 研究包生产

| 研究包 | 层级 | 输入 | 用途 |
| --- | --- | --- | --- |
| 物流研究包 | T3 | 铁板 + 铜线 | 物流工程 |
| 自动化研究包 | T4 | 齿轮 + 电路 | 自动化升级 |
| 冶金研究包 | T4 | 钢材 + 电路 | 冶金自动化 |
| 电子研究包 | T4 | 电路 x2 + 铜线 x2 | 高级电子学 |
| 机器人研究包 | T5 | 电机 + 轴承 | 机器人技术 |
| 核心研究包 | T5 | 伺服机构 + 处理器 | 自动化核心 |
| 效用研究包 | T6 | 自动化核心 + 伺服机构 + 处理器 | 规模化生产 |

研究包均由合成器生产。受研究锁定的研究包只会在其前置项目完成后出现在合成器双击配方面板中。

## 4. 研究中心机器

### 4.1 建造与连接

- 研究中心是独立机器，不再在左侧边栏展示研究栏目。
- 研究中心拥有四个输入端，没有输出端。
- 传送带可以从任意方向送入研究包。
- 输入缓存最多保存 48 个研究包，方便多条研究包总线汇入。

### 4.2 双击交互

双击研究中心打开研究项目面板。每个项目显示：

- 项目名称和效果。
- 对应研究包的层级图案。
- 已消耗数量 / 所需数量。
- 前置项目锁定状态。
- 当前研究、可选择和已完成状态。

选择新项目会重置机器当前加工 tick，但不会清空已经送入的研究包，也不会清除其他项目累计进度。

### 4.3 消耗规则

1. 未选择研究项目时，研究中心不接收任何研究包。
2. 前置项目未完成时，项目不可选择，也不会消耗缓存中的研究包。
3. 研究中心每完成一个项目节拍，只消耗一份当前项目需要的研究包。
4. 不属于当前项目的研究包保留在输入缓存中，不会被误吞。
5. 项目累计消耗达到要求后立即完成并发放解锁奖励。
6. 枢纽只统计最终产品交付，不再直接产生研究进度，防止绕过研究中心。

## 5. 研究项目树

| 项目 | 前置 | 研究包 | 数量 | 解锁 |
| --- | --- | --- | --- | --- |
| 物流工程 | 无 | 物流研究包 | 10 | 高速传送带 |
| 自动化升级 | 物流工程 | 自动化研究包 | 12 | 升级规划器、2 级机器 |
| 冶金自动化 | 自动化升级 | 冶金研究包 | 12 | 电机、轴承、钢制框架 |
| 高级电子学 | 自动化升级 | 电子研究包 | 14 | 处理器、机器人研究包 |
| 机器人技术 | 冶金自动化 + 高级电子学 | 机器人研究包 | 16 | 伺服机构、核心研究包 |
| 自动化核心 | 机器人技术 | 核心研究包 | 18 | 自动化核心、效用研究包 |
| 规模化生产 | 自动化核心 | 效用研究包 | 20 | 3 级机器 |

该研究树没有循环依赖：每个项目需要的研究包都能由当前或前置阶段已解锁的配方生产。

## 6. 推荐线路

### 6.1 前期物流研究

- 铁矿、煤矿 -> 铁锭 -> 铁板。
- 铜矿、煤矿 -> 铜锭 -> 铜线。
- 铁板 + 铜线 -> 物流研究包 -> 研究中心。
- 完成物流工程后，将主要总线升级为高速传送带。

### 6.2 中期自动化与冶金

- 铁板分流生产齿轮。
- 铁板与铜线生产电路。
- 齿轮 + 电路 -> 自动化研究包。
- 铁锭与煤矿生产钢材。
- 钢材 + 电路 -> 冶金研究包。

自动化升级与冶金自动化完成后，生产线可以开始制造电机、轴承和钢制框架。

### 6.3 后期电子与机器人

- 电路 + 铜线 -> 处理器与电子研究包。
- 电机 + 轴承 -> 机器人研究包。
- 电机 + 电路 + 轴承 -> 伺服机构。
- 伺服机构 + 处理器 -> 核心研究包。

### 6.4 终局自动化核心

- 处理器 + 钢制框架 + 伺服机构 -> 自动化核心。
- 自动化核心 + 伺服机构 + 处理器 -> 效用研究包。
- 效用研究包进入研究中心，完成规模化生产并开放 3 级机器。

## 7. 模块闭环

| 模块 | 输入 | 输出 | 下游 |
| --- | --- | --- | --- |
| 资源模块 | 固定 tick | T0 矿物 | 熔炉 |
| 冶金模块 | 矿物、煤矿 | T1 金属锭、T2 钢材 | 合成器 |
| 合成模块 | 多层材料、已解锁配方 | T2-T6 产品与研究包 | 物流、研究中心、枢纽 |
| 物流模块 | 机器输出 | 连续运输物品 | 生产机器、研究中心、枢纽 |
| 研究中心 | 所选项目、对应研究包 | 项目进度、解锁奖励 | 配方、物流、机器升级 |
| 编辑器模块 | 蓝图、框选、升级权限 | 大型工厂布局 | 模拟与渲染 |
| 性能模块 | 可见实体、帧时间 | 稳定 60 FPS 动画 | Canvas 双层渲染 |

## 8. 验收标准

- T0-T6 工业物品按照圆形、三角形、正方形、正五边形至正八边形显示。
- 合成器能生产轴承、钢制框架、处理器、伺服机构、自动化核心和七类研究包。
- 左侧边栏不再出现研究栏目。
- 研究中心可以放置并与传送带四向连接。
- 双击研究中心能够显示七个中文研究项目。
- 前置未满足的项目不可选择，且不会消耗研究包。
- 当前项目只消耗对应研究包，完成后正确解锁配方或机器能力。
- 枢纽交付不会绕过研究中心产生研究进度。
- 传送带动画继续保持双层 Canvas、连续插值和大工厂 60 FPS 优化。
- `npm test`、`npm run build`、`npm run lint` 与 `npm run test:e2e` 全部通过。
